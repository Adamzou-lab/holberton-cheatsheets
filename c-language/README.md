# C Language Cheatsheets

Cheatsheets interactives sur les concepts clés du C à Holberton School.

## Contenu

| Fichier | Sujet |
|---------|-------|
| [recursion_cheatsheet.html](./recursion_cheatsheet.html) | Récursion — cas de base, pile d'appels, exemples |
| [variadic_functions_cheatsheet.html](./variadic_functions_cheatsheet.html) | Fonctions variadiques — `va_list`, `va_start`, `va_arg` |
| [security_memory_lab_v2.html](./security_memory_lab_v2.html) | Sécurité mémoire — buffer overflow, pointeurs |

## Utilisation

Ouvrir directement dans un navigateur ou via un serveur local :

```bash
python3 -m http.server 8080
```

Puis aller sur `http://localhost:8080/`
